# 260. Lending place

```text
POST https://api.50x.cloud/json.place_lending
```

**Parameters:**
|Name|Type|Required|Description|
|---|---|:----------:|--------|
|sym|string|true|Asset symbol|
|amount|number|true|Value to lend|
|perc|number|true|Annual percent|

**Sample reply:**

```json
{"ok": true}
```
